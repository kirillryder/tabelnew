# Электронный Табель — Система учета рабочего времени

Веб-приложение для учета рабочего времени сотрудников с функциями управления сотрудниками, подразделениями и выплатой зарплат.

## 🚀 Технологии

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **Build Tool:** Vite

## 📋 Функционал

### 1. Табель (Таблица учета)
- Интерактивная сетка-календарь на выбранный месяц
- Строки — сотрудники, столбцы — дни месяца
- Двойной клик на ячейку для редактирования:
  - Выбор типа смены: Я (Явка), Б (Больничный), О (Отпуск), К (Командировка), Н (Неявка)
  - Ручной ввод часов
- Автоматический расчет:
  - Итого часов за месяц
  - Предварительная зарплата (часы × ставка)
- Выходные дни подсвечиваются красным
- Закрепленная первая колонка с ФИО при горизонтальном скролле
- Цветовое кодирование ячеек по типу смены

### 2. Сотрудники и Подразделения
**Подразделения:**
- Создание/удаление подразделений
- Иерархическая структура (подразделения могут иметь родительские)

**Сотрудники:**
- Добавление новых сотрудников с полями:
  - ФИО
  - Возраст
  - Подразделение
  - Почасовая ставка (₽/час)
  - Дата приема
- Редактирование данных сотрудника
- Удаление сотрудников

### 3. Ведомость Выплат
- Просмотр начисленной зарплаты за месяц
- Отображение выплаченных сумм
- Расчет задолженности к выплате
- Фиксация выплат с выбором метода:
  - Банковский перевод
  - Наличные
- История выплат по каждому сотруднику

### 4. Фильтрация и Поиск
- Поиск сотрудников по ФИО
- Фильтр по подразделениям
- Выбор месяца и года для просмотра

## 🔌 API (концептуально)

Полная спецификация REST API доступна в файле `API_SPEC.md`.

Основные эндпоинты:
- `GET/POST /employees` — управление сотрудниками
- `GET/POST /departments` — управление подразделениями
- `GET/POST /timesheet/entry` — записи табеля
- `GET/POST /payments` — выплаты
- `GET /reports/salary` — отчеты по зарплате

## 🎨 UI/UX Особенности

- **Минималистичный корпоративный дизайн** — чистый интерфейс как в современных CRM
- **Sticky колонка** — ФИО сотрудников всегда видно при скролле
- **Цветовое кодирование:**
  - Я (Явка) — синий фон
  - Б (Больничный) — красный фон
  - О (Отпуск) — зеленый фон
  - К (Командировка) — фиолетовый фон
  - Н (Неявка) — желтый фон
  - Выходные — серый фон
- **Адаптивная таблица** — горизонтальный скролл для больших месяцев
- **Модальные окна** — для добавления/редактирования данных
