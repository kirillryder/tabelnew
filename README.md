# Электронный Табель (Timesheet Application)

Веб-приложение для учета рабочего времени сотрудников с интерактивным календарем-таблицей.

## 🚀 Технологии

### Frontend
- **React 18** - UI библиотека с Hooks
- **TypeScript** - Типизация
- **Tailwind CSS** - Стилизация
- **Zustand** - State management
- **TanStack Table** - Headless таблица (архитектурная готовность)
- **Vite** - Сборщик

### Backend (концептуально)
- Node.js / FastAPI
- REST API спецификация в `API_SPEC.md`

## 📋 Функционал

- ✅ Динамическая таблица-календарь на выбранный месяц
- ✅ Подсветка выходных дней (суббота, воскресенье)
- ✅ Редактирование ячеек двойным кликом:
  - Выбор типа смены из выпадающего списка (Я, Б, О, К, Н)
  - Ввод количества часов вручную
- ✅ Авторасчет итогов:
  - Сумма часов за месяц
  - Расчет зарплаты на основе ставки сотрудника
- ✅ Фильтрация:
  - Поиск по ФИО
  - Фильтр по подразделениям
- ✅ Выбор месяца и года
- ✅ Sticky колонка с ФИО при горизонтальном скролле
- ✅ Цветовое кодирование типов смен

## 🛠️ Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## 🔌 API Integration

Для интеграции с бэкендом используйте спецификацию в файле `API_SPEC.md`.

Основные endpoints:
- `GET /api/v1/employees` - Список сотрудников
- `GET /api/v1/timesheet?year=&month=` - Данные табеля
- `POST /api/v1/timesheet/bulk` - Массовое обновление

## 📝 Пример использования

1. Выберите месяц и год в шапке
2. При необходимости отфильтруйте сотрудников по имени или подразделению
3. Двойной клик на ячейку для редактирования
4. Выберите тип смены или введите часы вручную
5. Итоги и зарплата рассчитываются автоматически
